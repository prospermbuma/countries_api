  ```json
    "countries": [
  ```

  ```json
  api/countries/all
  ```
  ::

  ::card{icon=noto:puzzle-piece}
  #title
  By random number.
  #description
  Get a random country from our countries list.
  <br>
  <br>
  ENDPOINT:
  ```
  api/countries/random
  ```
  ::
  ::card{icon=noto:puzzle-piece}
  #title
  By label.
  #description

  ```json
    "label": "Tanzania" //string
  }
  ```

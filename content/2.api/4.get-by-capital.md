# Get by capital

Method: <code>POST</code>

Find country by a label from our countries list.

```
api/countries/find
```
Body:

```
{
  "country": "Tanzania" //string
}
```
Response:

```
{
  "countries": [
    {
      "_id": "TZ",
      "label": "Tanzania, United Republic of Tanzania"
    }
  ]
}
```
